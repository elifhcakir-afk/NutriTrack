🥗 NutriTrack – Calorie & Macro Tracker

A multilingual web application designed to calculate daily calorie needs and track consumed foods including automatic macronutrient estimation.
The project was developed based on personal interest in fitness and nutrition, focusing on structured data processing and logical calculation models.

🌍 Language Support / Sprachen

🇹🇷 Turkish (Türkçe)
🇬🇧 English
🇩🇪 German (Deutsch)

🚀 Key Features

1️⃣ Daily Calorie Calculator
Calculates daily calorie needs based on:
Age
Height
Weight
Gender
Activity level
Uses a structured calculation model (Harris-Benedict formula logic)

2️⃣ BMI Calculator
Automatic BMI calculation
Displays weight category:
Underweight
Normal
Overweight
Obese

3️⃣ Food Tracking System
Users can manually enter consumed foods
Automatically estimates:
Calories
Protein
Carbohydrates
Fat
Daily food intake is saved using LocalStorage

4️⃣ Data Persistence
All user data is stored in LocalStorage
No backend required
Data remains saved even after closing the browser

🛠️ Technical Stack

HTML5
CSS3
JavaScript (ES6)
LocalStorage API

No external frameworks used.
Pure frontend logic implementation.
